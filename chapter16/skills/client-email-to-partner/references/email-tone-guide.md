# 거래처 이메일 문체 기준

## 기본 원칙

- 첫 문장은 정중한 인사로 시작한다.
- 요청사항은 bullet로 분리한다.
- 회신 기한은 명확하게 적는다.
- 첨부 파일이 없거나 불명확하면 `확인 필요`로 표시한다.
- 상대가 해야 할 행동을 마지막에 다시 적는다.

## 표현 예시

| 목적 | 제목 예시 |
| --- | --- |
| 확인 요청 | `[확인 요청] 자료 검토 요청` |
| 일정 조율 | `[일정 조율] 회의 가능 시간 확인` |
| 자료 송부 | `[자료 송부] 프로젝트 산출물 전달` |
| 후속 안내 | `[후속 안내] 회의 결과 공유` |

## 피해야 할 표현

- `빨리 처리 바랍니다`
- `아무거나 보내주세요`
- `대충 확인 부탁드립니다`
- 배경 없이 요청만 쓰는 문장

# 팀 공용 프롬프트 라이브러리

## 1. 코드 리뷰
> 이 변경분을 보안·성능·가독성 순으로 리뷰해 줘. 위험도 ★~★★★로 표시.

## 2. 회의록 정리
> 회의록을 결정사항·액션 아이템(담당자/마감일)·확인 필요 항목으로 정리해 줘.

## 3. 커밋 메시지
> 변경 파일을 보고 Conventional Commits 규격으로 커밋 메시지를 작성해 줘.

## 4. 테스트 생성
> 이 함수의 엣지 케이스 5개 + 회귀 테스트 + 통합 테스트를 만들어 줘.

## 5. 릴리스 노트
> git 로그를 사용자 친화적 릴리스 노트로 변환해 줘. 기능/수정/개선으로 분류.

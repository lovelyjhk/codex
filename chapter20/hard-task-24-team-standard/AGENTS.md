# 팀 작업 표준 (AGENTS.md)

## 코딩 규칙
- 모든 함수에 한국어 docstring을 단다.
- 파일 인코딩은 UTF-8로 고정한다.
- 외부 라이브러리 추가 시 이유를 커밋 메시지에 적는다.

## 금지 사항
- `danger-full-access` 권한을 자동화 스크립트에 사용하지 않는다.
- API 키·비밀번호를 코드나 프롬프트에 직접 넣지 않는다.
- main 브랜치에 직접 푸시하지 않는다 (PR 필수).

## 테스트
- 변경 후 `pytest` 통과를 확인한다.
- 새 기능에는 엣지 케이스 테스트를 1개 이상 추가한다.

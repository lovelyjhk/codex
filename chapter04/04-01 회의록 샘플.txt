프로젝트 정기 점검 회의록

1. 회의 개요
- 회의명: 차세대 업무 시스템 구축 프로젝트 주간 점검
- 일시: 2026-05-16 10:00~11:00
- 장소: 온라인 회의
- 참석자: 김민수 PM, 이지현 개발리드, 박서연 QA리드, 최준호 인프라 담당, 정유진 현업 담당
- 작성자: 확인 필요

2. 회의 목적
- 개발 진행 현황, 테스트 준비 상황, 인프라 이슈, 오픈 일정 리스크를 점검한다.
- 지연 가능성이 있는 과제를 식별하고 후속 조치를 확정한다.

3. 주요 논의 내용
- 개발리드는 핵심 업무 기능 12개 중 9개 기능 개발이 완료되었고, 나머지 3개 기능은 화면 연계와 권한 처리 작업이 남아 있다고 보고했다.
- QA리드는 통합 테스트 시나리오 초안은 작성되었으나 현업 검토가 완료되지 않아 테스트 착수 일정에 영향이 있을 수 있다고 설명했다.
- 인프라 담당자는 테스트 서버의 스토리지 증설 작업이 필요하며, 현재 용량으로는 통합 테스트 중 로그 적재에 제약이 있을 수 있다고 공유했다.
- 현업 담당자는 권한 매트릭스 기준이 일부 부서에서 아직 확정되지 않았다고 보고했다.
- PM은 개발 완료 기준과 테스트 착수 기준을 명확히 분리하고, 미확정 항목은 별도 이슈 목록으로 관리하자고 제안했다.

4. 결정사항
- 통합 테스트 착수 기준은 핵심 기능 개발 완료와 QA 시나리오 현업 검토 완료로 정한다.
- 권한 매트릭스 미확정 부서는 별도 이슈 목록으로 관리한다.
- 테스트 서버 스토리지 증설은 통합 테스트 착수 전 완료한다.
- 잔여 개발 기능 3개는 개발리드가 우선순위를 조정해 이번 주 내 완료 계획을 공유한다.

5. 액션 아이템
- 이지현 개발리드는 잔여 개발 기능 3개의 완료 계획과 상세 일정을 2026-05-18까지 PM에게 공유한다.
- 박서연 QA리드는 통합 테스트 시나리오 초안을 현업 담당자에게 전달하고 검토 의견을 2026-05-20까지 취합한다.
- 정유진 현업 담당자는 부서별 권한 매트릭스 미확정 항목을 정리해 2026-05-21까지 PM에게 전달한다.
- 최준호 인프라 담당자는 테스트 서버 스토리지 증설 가능 일정과 작업 영향도를 2026-05-19까지 공유한다.
- 김민수 PM은 미확정 이슈 목록을 작성하고 차주 회의에서 진행 상태를 점검한다.

6. 리스크 및 이슈
- 통합 테스트 시나리오 현업 검토가 지연될 경우 테스트 착수 일정이 지연될 수 있다.
- 권한 매트릭스 확정이 늦어질 경우 권한 처리 개발과 테스트 범위 확정에 영향이 있다.
- 테스트 서버 스토리지 증설이 통합 테스트 전 완료되지 않으면 로그 적재 및 테스트 수행에 제약이 발생할 수 있다.
